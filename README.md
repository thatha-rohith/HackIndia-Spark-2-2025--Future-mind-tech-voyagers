- Node.js & npm

### Installation
```sh
# Clone the repository
git clone https://github.com/thatha-rohith/HackIndia-Spark-2-2025--Future-mind-tech-voyagers.git
```

```sh

npx hardhat compile

# Deploy smart contracts
```

